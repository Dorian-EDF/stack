# stack

## Objectifs de l'outil
projet multi-package pour analyser facilement des datas

## Installation
```bash
git clone https://github.com/EDF-Gestion/stack.git
cd stack
uv sync
```

## Utilisation
```bash
uv run stack [arguments]
```

## Configuration
Variables d'environnement nécessaires (fichier `.env`) :
- `API_KEY` : clé pour le service X
